# PUF Anomaly Detection for IoT Security

## Project Overview
This project implements a lightweight AI-assisted layer to protect XOR Arbiter PUFs from Machine Learning modeling attacks.

## How it Works
1. **XOR Arbiter PUF**: Simulates 64-bit hardware with environmental noise.
2. **Behavioral Monitoring**: Analyzes the statistical "texture" of authentication sessions.
3. **Isolation Forest**: Detects anomalies in response bias and variance to flag attackers.

## Results
The system successfully identifies modeling attacks as statistical outliers, as shown in the generated scatter plots.
